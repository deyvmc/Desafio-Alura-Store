# Desafio-Alura-Store
Este proyecto consiste en el análisis de datos de ventas de cuatro tiendas pertenecientes a la cadena Alura Store.
El objetivo del análisis es ayudar al Sr. João a tomar una decisión estratégica sobre cuál de sus tiendas debería vender para invertir en un nuevo emprendimiento.

Para ello, se analizaron distintos aspectos del rendimiento de cada tienda utilizando Python.

Objetivos del análisis

El análisis se basó en cinco métricas principales:

Facturación total de cada tienda.

Categorías de productos más vendidas.

Calificación promedio otorgada por los clientes.

Productos más y menos vendidos.

Costo promedio de envío.

A partir de estas métricas se buscó identificar cuál de las tiendas presenta el rendimiento más bajo.

Herramientas utilizadas

Las principales herramientas utilizadas para el desarrollo del análisis fueron:

Python

Pandas para manipulación y análisis de datos

Matplotlib para visualización de datos

Google Colab para el desarrollo del notebook

Visualización de datos

Durante el análisis se generaron diferentes gráficos para facilitar la interpretación de los resultados, incluyendo:

Gráficos de barras para comparar facturación entre tiendas.

Gráficos de ventas por categoría.

Comparación de calificaciones promedio.

Análisis de costos de envío.

Estas visualizaciones permiten observar de manera más clara las diferencias de rendimiento entre las tiendas.
